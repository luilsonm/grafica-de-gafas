Sistemas Embebidos

Fundamentalmente muestra la hora de forma continua y es utilizado para medir el tiempo y organizar actividades según horarios establecidos.

Función que se utiliza para medir intervalos de tiempo con precisión, empezando desde cero y avanzando en cuenta ascendente.

Funcion para medir o controlar el tiempo transcurrido, con el fin de realizar una acción específica después de un período predeterminado.

Captura imágenes fijas o vídeos mediante la captación de la luz sobre un sensor fotosensible, como una película fotográfica o un sensor digital, representación visual que puede ser almacenada o mostrada.
